AI Equity Analyst

A multi-step LLM pipeline that turns a stock ticker into a one-page investment briefing — live fundamentals and news, synthesized by three chained GPT calls.

View on GitHub See a sample report

How it works

Every LLM call is stateless, so each stage's output is passed into the next. Two extraction stages feed a final synthesis stage that reconciles the numbers with the narrative.

ticker
  |
  |-- [1] Fundamental analyst   <- yfinance .info   (valuation, margins, growth, balance sheet)
  |         notes: Strengths / Concerns / Metrics to watch
  |
  |-- [2] News analyst          <- yfinance .news   (recent headlines + summaries)
  |         notes: Tone / Key developments / Risks
  |
  |-- [3] Senior analyst        <- combines notes [1] + [2]
            final report: Snapshot, Fundamentals, News, Fundamentals vs News,
            Outlook by Horizon, Bottom Line   (streamed live)
